Rat hat base for silicon probe implantation.

Assembly instructions can be found in the Assembly_instructions_rat_base_plate.pdf file. 

The .stl files are optimized for Formlabs Form2 printer and Grey-v4/Clear-v4 Resin. For other printers and/or resins, some modifications may be necessary.

1 version is made available here: hat base for rats.stl
