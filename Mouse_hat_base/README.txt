Mouse hat base for silicon probe implantation.

Assembly instructions can be found in the Assembly_instructions_mouse_base_plate.pdf file. 
Surgery video using this design is available here: https://www.youtube.com/watch?v=NjBai0FpuOE

The .stl files are optimized for Formlabs Form2 printer and Grey-v4/Clear-v4 Resin. For other printers and/or resins, some modifications may be necessary.

1 version is made available here: Hat_for _mouse_inside_10.6mmx8.stl
